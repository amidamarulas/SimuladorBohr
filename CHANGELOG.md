# Changelog

Este archivo documenta los cambios realizados en cada versión de **SimuladorBohr**.

---

## [0.1.0] - 2025-10-01
### Added
- Implementación inicial del paquete.
- Módulo `constants.py` con constantes físicas fundamentales.
- Módulo `energy.py` con funciones para cálculo de energías en J y eV.
- Módulo `radius.py` para cálculo de radios orbitales.
- Módulo `transitions.py` para energías, frecuencias y longitudes de onda de transiciones.
- Módulo `plotting.py` para visualización de niveles de energía y órbitas.
- Ejemplo completo en `examples/example_usage.py`.
- Archivos iniciales: `pyproject.toml`, `.gitignore`, `README.md`, `LICENSE`.

---

## Formato

El proyecto sigue un estilo de cambios basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).  
Las categorías principales son:
- **Added**: Nuevas funcionalidades.  
- **Changed**: Cambios en funcionalidades existentes.  
- **Deprecated**: Funcionalidades obsoletas que se eliminarán.  
- **Removed**: Eliminaciones.  
- **Fixed**: Corrección de errores.  
- **Security**: Cambios relacionados con vulnerabilidades de seguridad.
